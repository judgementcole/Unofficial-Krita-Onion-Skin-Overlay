# Unofficial-Krita-Onion-Skin-Overlay
Modified version of Krita where Onion Skin is rendered on top instead of bottom of the drawing/painting. Made to make animating in Krita easier
<br>
<br>
<br>
## How to use
1. Download the latest release from the releases tab
2. Extract
3. Navigate to 'Krita_OnionSkinOverlay_vX.X.X/bin'
4. Run Krita.exe
